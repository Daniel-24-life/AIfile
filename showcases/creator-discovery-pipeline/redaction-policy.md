# Redaction Policy

This folder is a public version of private creator discovery work.

## Included

The public folder includes:

- workflow structure
- system architecture
- review gates
- evidence model
- fake sample rows
- small code skeleton with fake data

## Outside The Public Folder

The following materials stay private:

- real creator names and handles
- real emails, DMs, profile links, and contact pages
- original workbooks
- platform sessions and browser profiles
- cookies, tokens, keys, and credentials
- exact search terms and seed lists
- exclusion lists
- platform automation details
- private thresholds and scoring rules
- full scripts and full SOPs
- client, employer, partner, or campaign details
- original logs, screenshots, and exports

## Generalization

The public version generalizes:

- platform-specific collection into candidate discovery
- real profile fields into fake schema fields
- private review rules into public review gates
- workbook outputs into sample artifact categories
- business-facing decisions into review states

## Pre-Upload Check

Before upload, this package should contain:

- no real handles
- no real contact fields
- no real profile URLs
- no account sessions
- no workbook exports
- no original prompts
- no full scraping workflow
- no operational recipe that can reproduce the private project
