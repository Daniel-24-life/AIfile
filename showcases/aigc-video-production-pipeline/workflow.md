# Workflow

This workflow describes the public shape of the project. It is intentionally abstract, but it still reflects the actual work pattern: creative planning, asset tracking, review, and package preparation.

## Stage 1: Creative Brief

The work starts with a clear creative brief:

- story premise
- audience
- desired tone
- visual direction
- delivery requirements
- review standard

The goal is to make the project inspectable before generation begins. A weak brief makes later visual review subjective and slow.

## Stage 2: Shot Plan

The story is broken into production units. A unit can be a scene, shot, or other reviewable segment.

Each unit needs:

- narrative role
- visual intent
- required assets
- review status
- connection to nearby units

This keeps the project from becoming a pile of disconnected images or clips.

## Stage 3: Prompt Archive

Prompt work is treated as production documentation.

The archive tracks:

- prompt purpose
- reference relationship
- asset role
- revision status
- review notes

Original prompts stay outside this folder. The note here is about why prompt tracking mattered: without a record, a selected asset can become hard to explain, revise, or compare.

## Stage 4: Visual Asset Review

Generated visuals are reviewed before being used downstream.

Review focuses on:

- character continuity
- scene consistency
- composition
- narrative fit
- visible defects
- asset readiness

This stage separates candidate assets from selected assets.

## Stage 5: Review Package

The project then becomes a review package. This is the bridge between creative production and final handoff:

- production notes
- shot table
- image contact sheets
- document drafts
- spreadsheet summaries
- visual QA previews

The review package makes it possible to inspect the project as a system.

## Stage 6: Submission Readiness

Final readiness is checked across:

- file completeness
- naming consistency
- document readability
- visual previews
- missing information
- release boundary

The key idea is simple: creative output still needs production control.
