# Concept Card: Prompt Archive As Production Record

## Insight

Prompt work should be treated as production documentation.

## Meaning

In a multi-shot AIGC project, prompts are not disposable chat messages. They record intent, references, revisions, and asset roles.

## Rule

Track prompts by production unit:

```text
unit -> intent -> reference role -> output candidate -> review status
```

## Reuse

Use this card for image generation, video generation, storyboard work, and any creative project with repeated AI generation steps.
