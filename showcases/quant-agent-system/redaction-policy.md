# Redaction Policy

This public package is intentionally sanitized.

## Excluded

The following private materials are excluded:

- raw data
- cached market data
- experiment outputs
- logs
- credentials
- exchange-specific private code
- strategy formulas
- candidate pools
- backtest results
- validation result tables
- system dialogue history
- internal task backlog
- private business methodology

## Generalized

The following concepts are generalized:

- domain-specific names
- strategy references
- target metrics
- execution venue details
- candidate identifiers
- run identifiers
- private artifact paths

## Included

The following public-safe materials are included:

- architecture narrative
- governance principles
- evidence lifecycle
- dry-run execution pattern
- illustrative policy checks
- representative tests

## Review Requirement

This package should be manually reviewed before publication.

Recommended checks:

- no credentials
- no private path leakage
- no strategy logic
- no real results
- no private dataset names
- no misleading production claims
