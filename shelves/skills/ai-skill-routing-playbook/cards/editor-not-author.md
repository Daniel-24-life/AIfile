# Concept Card: Style Cleanup Is Editing

## Insight

Style cleanup is an editor role.

## Meaning

Style cleanup should remove generic or machine-like phrasing while preserving facts, intent, and the author's voice. It should not invent the thesis, evidence, story, or personal experience.

## Rule

Use anti-AI-writing cleanup near the end:

```text
draft -> user judgment -> review -> style cleanup -> final check
```

## Reuse

Use this card for public writing, portfolio notes, article drafts, and project READMEs.
