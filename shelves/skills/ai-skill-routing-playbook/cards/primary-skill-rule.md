# Concept Card: Primary Skill Rule

## Insight

Most tasks should start with one primary skill.

## Meaning

If several tools enter at once, it becomes hard to see who is drafting, reviewing, verifying, or publishing. A primary skill gives the task a clear owner.

## Rule

Choose the primary skill by deliverable:

```text
article -> writing
review -> reviewer
file -> document tool
evidence -> verification
publish -> release workflow
```

## Reuse

Use this card before complex writing, document, research, GitHub, and automation tasks.
