# Creator Discovery Skeleton

This is a small public code skeleton for creator discovery governance.

It includes:

- fake creator records
- controlled review states
- simple deduplication checks
- evidence completeness checks
- boundary tests

Outside this public skeleton:

- real collection scripts
- platform automation
- browser profiles
- real handles and contact fields
- exact search terms and seed lists
- scoring thresholds
- workbook exports

## Run Tests

```powershell
python -m pytest shelves\code\creator-discovery-skeleton\tests
```
