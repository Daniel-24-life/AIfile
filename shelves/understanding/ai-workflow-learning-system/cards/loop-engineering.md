# Concept Card: Loop Engineering

## One-line Insight

Harness 解决一次任务怎么做，Loop 解决任务如何持续转起来。

## Meaning

一个任务值得建 Loop，至少要满足：重复发生、能验证、成本可承受、AI 有足够工具和权限。

## My Rule

先做半自动陪跑，再决定是否 Loop 化。

## Four-gate Test

```text
1. 它会重复发生吗？
2. 验证能自动或半自动完成吗？
3. 每轮运行成本可接受吗？
4. AI 是否有足够工具权限完成闭环？
```

## Loop Actions

- Discover: 发现该做的事
- Deliver: 隔离交付
- Verify: 独立验证
- Persist: 状态持久化
- Schedule: 调度下一轮

## Reuse

适用于周期性检查、资料更新、日志扫描、项目健康巡检、知识库维护。
