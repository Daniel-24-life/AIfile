# Concept Card: Skill Candidate Rule

## One-line Insight

重复 3 次以上、流程稳定、验收清楚的任务，才值得 Skill 化。

## Meaning

Skill 的价值是降低重复任务的启动成本和错误率。

## My Rule

当同类任务反复出现时，先沉淀模板和 checklist；稳定后再升级成 Skill。

## Skill Candidate Checklist

- 同类任务出现 3 次以上
- 输入材料结构相似
- 输出格式相似
- 错误模式可预期
- 验收方式清楚
- 流程能写成短规则

## Do Not Skill

- 一次性任务
- 目标还不稳定
- 主要依赖临场判断
- 验收方式说不清
