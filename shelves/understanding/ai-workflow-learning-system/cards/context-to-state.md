# Concept Card: Context To State

## One-line Insight

长任务不能靠聊天记忆硬撑，要把状态写到文件里。

## Meaning

上下文窗口会稀释、腐烂和遗忘。真正稳定的是外部状态：spec、handoff、README、checklist、测试、报告和目录结构。

## My Rule

任何跨天、跨轮、跨文件或跨任务的工作，都必须有状态落盘。

## State Assets

- task spec
- current status
- decision record
- handoff
- verification report
- reusable template
- next action

## Reuse

适用于长程项目、代码修改、研究整理、作品集维护、自动化流程。
