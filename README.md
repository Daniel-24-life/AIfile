# AIfile

AIfile is a curated library for code, skills, articles, project results, notes, and working ideas.

This repository is managed as a library, not as a raw file dump. The goal is to collect only useful artifacts, place them on the right shelf, and give each item enough context to be understood later.

## Shelves

| Path | Use |
| --- | --- |
| `inbox/` | Temporary landing area for unsorted items. |
| `shelves/code/` | Code, scripts, prototypes, and engineering results. |
| `shelves/skills/` | Codex skills, prompts, agent workflows, reusable procedures. |
| `shelves/articles/` | Essays, article drafts, public writing, polished notes. |
| `shelves/understanding/` | Concept maps, frameworks, reading notes, and personal knowledge structures. |
| `showcases/` | Finished or presentable work worth sharing. |
| `catalog/` | Indexes, tags, metadata, and librarian notes. |

## Rule

Do not upload everything by default. Select, summarize, classify, then publish.

Each meaningful item should have:

- title
- type
- source or origin
- status
- short abstract
- tags
- next action

## Current Status

The repository is initialized as a library shell. Content will be added deliberately, one curated batch at a time.
